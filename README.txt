Só é garantido que o código funcione bem em SWI-Prolog
$ swipl

Para poder começar a testar frases basta consultar consultar o ficheiro grammar:
?- [grammar].

Para testar os exemplos dados na proposta de trabalho pode-se consultar tambem o ficheiro dutra_examples:
?- [grammar,dutra_examples].

Podendo-se chamar os predicados:

correct/0.

incorrect/0.
